# API/Sistema de Gestão Farmáceutica

## Pré-requisitos

-nodeJs v18+
-typeScript

## Instalação

git clone https://github.com/tiagosantostsdev/farm_server
cd seu-repositorio
npm install


## Utilização

-npm run dev / para rodar a dist em javaScript
-npm start / para rodar o typeScript
